Welcome to the Real Life Gaming System

The RLGS is a free, electronic gaming system to be used with tactical sports like paintball, airsoft, nerf, larp.

Additional material

In this project I used other people's work, which was kindly released under creative commons or open source license terms.
  • The Spring Framework is the basis on which the RLGS is built upon. It is an application framework and inversion of control container for the Java platform.
  • Zeus - created by Freepik - Flaticon
  • Buzzer - created by Freepik - Flaticon
