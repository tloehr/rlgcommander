Willkommen zum Real Life Gaming System

The RLGS is a free, electronic gaming system to be used with tactical sports like paintball, airsoft, nerf, larp.

Zusätzliches Material

In diesem Projekt habe ich auch fremdes Material verwendet, das von anderen Personen freundlicherweise als gemeinfrei oder unter einer freien Lizenz veröffentlich wurde.
  • Das Spring Framework ist die Grundlage auf der das RLGS aufbaut. Es ist eine Applikations Plattform und ein inversion of Control Container für Java.
  • Zeus - created by Freepik - Flaticon
  • Buzzer - created by Freepik - Flaticon
